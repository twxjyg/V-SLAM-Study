# V-SLAM-Study
## 这是什么?

该项目是参加深蓝学院的VSLAM课程过程中完成的作业以及相关课程资料的集合

## 编辑工具
所有markdown文件使用VS Code编辑，并安装了插件:Markdown Preview Enhanced
->[https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced]

该插件对集成了很多有用的markdown插件：
1. Math Typing
2. Diagrams
3. TOC
4. Matplotlib(直接运行python代码b并可视化结果)

还有很多其他有用插件，各位客官可移步其官方文档探究。
